## I. Mục tiêu
Rèn luyện kỹ năng kết nối thiết bị ngoại vi với máy tính, gồm kết nối vật lý, cài driver (nếu cần), cấu hình và kiểm tra hoạt động.

## II. Thiết bị đã thực hành
1. Màn hình phụ (External Monitor) qua cổng HDMI.
2. Tai nghe không dây (Bluetooth Headphones).
3. Điện thoại thông minh (Redmi Note 11T / iPhone 11) để truyền dữ liệu.

## III. Quy trình kết nối chi tiết

### 1. Màn hình phụ qua HDMI
- **Kết nối vật lý:** Cắm cáp HDMI từ laptop sang màn hình, bật màn hình, chọn đúng cổng HDMI (Input).
- **Driver:** Windows tự nhận; nếu lỗi thì cập nhật driver card đồ họa trong Device Manager.
- **Cấu hình thiết bị:** Vào `Settings > System > Display`, chọn `Extend these displays`, chỉnh độ phân giải khuyến nghị.
- **Kiểm tra hoạt động:** Kéo cửa sổ qua lại giữa 2 màn hình, kiểm tra hình ảnh ổn định.

### 2. Tai nghe Bluetooth
- **Kết nối vật lý:** Sạc pin tai nghe, bật nguồn, đưa tai nghe vào chế độ Pairing.
- **Driver:** Kiểm tra adapter Bluetooth trong Device Manager, cập nhật qua Windows Update nếu cần.
- **Cấu hình thiết bị:** Vào `Settings > Bluetooth & devices > Add device > Bluetooth`, chọn tên tai nghe và kết nối.
- **Kiểm tra hoạt động:** Phát nhạc/video và thử micro để kiểm tra âm thanh.

### 3. Điện thoại truyền dữ liệu (Redmi Note 11T / iPhone 11)
- **Kết nối vật lý:** Cắm cáp USB điện thoại với máy tính.
- **Driver:**  
  - Redmi: Windows thường tự nhận MTP.  
  - iPhone: Cài iTunes/Apple Devices để nhận driver Apple Mobile Device (nếu cần).
- **Cấu hình thiết bị:**  
  - Redmi: Chọn chế độ `Truyền tệp (MTP)`.  
  - iPhone: Chọn `Trust This Computer` và nhập mật mã.
- **Kiểm tra hoạt động:** Sao chép thư mục ảnh/video từ điện thoại sang máy tính và mở file kiểm tra.

## IV. Vấn đề gặp phải và cách khắc phục
1. **Màn hình ngoài không nhận tín hiệu:** Kiểm tra lại cáp HDMI, chọn đúng Input, nhấn `Win + P` chọn `Extend`.
2. **Tai nghe không hiện trong danh sách Bluetooth:** Đưa tai nghe về Pairing lại, xóa thiết bị cũ rồi ghép nối lại.
3. **Điện thoại không hiện bộ nhớ:**  
   - Redmi: Chuyển đúng chế độ MTP.  
   - iPhone: Bấm Trust và cài driver Apple.

## V. Hướng dẫn ngắn (infographic/video 2-3 phút)
**Chủ đề:** Kết nối tai nghe Bluetooth với laptop  
1. Bật Bluetooth trên laptop.  
2. Bật tai nghe, vào Pairing.  
3. Chọn Add device và chọn tên tai nghe.  
4. Đặt tai nghe làm thiết bị âm thanh mặc định.  
5. Phát nhạc để kiểm tra.
